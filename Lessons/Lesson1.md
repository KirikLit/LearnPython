###Переменные  

Переменные - это такие ячейки, в которых хранятся данные
>_В переменных могут храниться любые типы данных: строки, числа и т. д._

Пример:
```python
boolean = True
integer = 15
string = 'Hello'

print(boolean)
print(integer)
print(string)
```
На выходе мы получим:
```doctest
True
15
Hello
```
####Математические операции
Математические операции - очень важная часть языка. Без них не обойдётся ни одна программа. В языке Python
имеются разные математические операторы. Вот некоторые из них:
    
    Простые математические операторы
    * - Умножение
    ** - Возведение в степень
    + - Сложение
    - - Вычитание
    / - Деление

Также существую специальные операторы для переменных:
    
    x - Переменная, n - любое число
    x += n  # Эквивалент x = x + n
    x -= n  # Эквивалент x = x - n  
    x *= n  # Эквивалент x = x * n
    x /= n  # Эквивалент x = x / n

Пример: 
```python
x = 10
y = 20
z = 5

y += z          # y = y(20) + z(5) = 25
z *= 2          # z = z(5) * 2 = 10

print(x + y)    # переменная x(10) + переменная y(25)
print(z ** 2)   # переменная z(10) во второй степени
print(z)        # переменная z
```
Вывод программы из примера:
```doctest
35
100
10
```
>Со строками нельзя проводить математические операции, но для них есть функция eval()

```python
# Пример использования функции eval()
    
x = '2 + 2 * 2'
print(eval(x))
```
На выходе мы получим:
    
    6

###Типы данных
Мы уже изучили переменные, но перед вводом и выводом данных я хочу рассказать про типы данных  

В Python существует много типов данных:
```
str - строка 
'Hello' - строка

bool - булевое значение (True или False)
True - bool

int - целое число
16 - целое чило
'16' - строка

float - вещественное число
1.6 - вещественное число
```
>В пайтоне существуют и другие типы, такие как списки, словари и т.д., но мы их пройдём позже

Напишем программу, которая покажет типы данных
```python
type1 = True
type2 = 1.6
type3 = 16
type4 = '16'
type5 = 'Hello'

print(type(type1))
print(type(type2))
print(type(type3))
print(type(type4))
print(type(type5))
```
Посмотрим, что мы увидели на выходе:
```doctest
<class 'bool'>
<class 'float'>
<class 'int'>
<class 'str'>
<class 'str'>
```

###Ввод и вывод данных
Теперь можно приступать к изучению ввода и вывода данных  

Вывод данных осуществляется командой print():
    
    print('Hello World!')

> Нельзя выводить неизвестный для языка объект, например:  
> print(Hello)  
> Правильно будет  
> print('Hello')

####Вывод данных
Ввод данных осуществляется командой input(текст при вводк)  

Например:
```python
x = input('Введите что-нибудь: ')
print(x)
```
> Данная программа потребует от вас ввести что-либо, а затем выведет это на экран  

> Команда ввода возвращает строку. Если надо число, то используйте: 
> int(input())
>> Аналогично и для других типов данных
